# KirayaPe 🏠

> Crafted for the modern renters.

**KirayaPe** is a premium rental platform designed to simplify the rental experience for tenants and landlords in India. Discover zero-brokerage PGs, flats, and houses, automate rent payments, earn rewards, and boost your credit score.

---

## ✨ Features

- 🔍 **AI Flat/PG Finder**: Search and find zero-brokerage rental properties matching your preferences.
- 💳 **Auto-Pay Rent**: Automate rent payments effortlessly with integrated AI.
- 🎁 **Rent Rewards**: Earn cashback and premium rewards every time you pay rent.
- 💰 **Rent Loans**: Get instant credit/loans specifically for rental deposits and payments.
- 📈 **Credit Score Booster**: Report rent payments to credit bureaus to build and increase your credit rating.
- 🧑‍💼 **Landlord Dashboard**: Dedicated panel for landlords to manage listings, collect rent, and check tenant applications.

---

## 🛠️ Tech Stack & Structure

- **Core**: HTML5, Vanilla CSS3 (Custom design system), JavaScript (ES6)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Configuration**:
  - `vite.config.js`: Setup for clean URLs and production building.
  - `vercel.json`: Native Vercel deployment with clean URL routing enabled.

## 🚀 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Ankushpanday1/KirayaPe.git
   cd KirayaPe
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
